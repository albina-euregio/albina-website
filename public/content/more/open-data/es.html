Avalanche.report XML

El boletín diario se publica como XML y se puede encontrar aquí:

XML

Every bulletin is published in 7 langugages and the corresponding XML files for the most recent bulletin are available at this URLs:

  • DE: https://avalanche.report/albina_files/latest/de.xml
  • IT: https://avalanche.report/albina_files/latest/it.xml
  • EN: https://avalanche.report/albina_files/latest/en.xml
  • FR: https://avalanche.report/albina_files/latest/fr.xml
  • SP: https://avalanche.report/albina_files/latest/sp.xml
  • CA: https://avalanche.report/albina_files/latest/ca.xml
  • OC: https://avalanche.report/albina_files/latest/oc.xml
