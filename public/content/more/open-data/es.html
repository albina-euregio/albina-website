Avalanche.report JSON and XML

The daily bulletin is published as JSON and XML and can be found here:

CAAML v6 JSON

Cada boletín se publica en 7 idiomas y los archivos JSON correspondientes para el boletín más reciente están disponibles en estas URL:

  • DE: https://static.avalanche.report/bulletins/latest/EUREGIO_de_CAAMLv6.json
  • IT: https://static.avalanche.report/bulletins/latest/EUREGIO_it_CAAMLv6.json
  • EN: https://static.avalanche.report/bulletins/latest/EUREGIO_en_CAAMLv6.json
  • FR: https://static.avalanche.report/bulletins/latest/EUREGIO_fr_CAAMLv6.json

  • CA: https://static.avalanche.report/bulletins/latest/EUREGIO_ca_CAAMLv6.json
  • OC: https://static.avalanche.report/bulletins/latest/EUREGIO_oc_CAAMLv6.json

The JSON files follow the CAAML v6 standard. Download JSON schema file.
