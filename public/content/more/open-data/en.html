Avalanche.report JSON and XML

The daily bulletin is published as JSON and XML and can be found here:

CAAML v6 JSON

Every bulletin is published in 7 languages and the corresponding JSON files for the most recent bulletin are available at this URLs:

  • DE: https://static.avalanche.report/bulletins/latest/de_CAAMLv6.json
  • IT: https://static.avalanche.report/bulletins/latest/it_CAAMLv6.json
  • EN: https://static.avalanche.report/bulletins/latest/en_CAAMLv6.json
  • FR: https://static.avalanche.report/bulletins/latest/fr_CAAMLv6.json
  • ES: https://static.avalanche.report/bulletins/latest/es_CAAMLv6.json
  • CA: https://static.avalanche.report/bulletins/latest/ca_CAAMLv6.json
  • OC: https://static.avalanche.report/bulletins/latest/oc_CAAMLv6.json

The JSON files follow the CAAML v6 standard. Download JSON schema file.
