licenza

All our software is open source. The corresponding licenses can be found directly in the projects. All data provided, such as the avalanche report, weather station data, etc., are licensed under CC BY 4.0 DEED.

Valanghe.report JSON e XML

Il bollettino quotidiano è pubblicato in formato JSON e XML e si trova qui:

CAAML v6 JSON

Ogni bollettino è pubblicato in 7 lingue e il corrispondente JSON I file per il bollettino più recente sono disponibili a questo URL:

  • DE: https://static.avalanche.report/bulletins/latest/EUREGIO_de_CAAMLv6.json
  • IT: https://static.avalanche.report/bulletins/latest/EUREGIO_it_CAAMLv6.json
  • EN: https://static.avalanche.report/bulletins/latest/EUREGIO_en_CAAMLv6.json
  • FR: https://static.avalanche.report/bulletins/latest/EUREGIO_fr_CAAMLv6.json
  • ES: https://static.avalanche.report/bulletins/latest/EUREGIO_es_CAAMLv6.json
  • CA: https://static.avalanche.report/bulletins/latest/EUREGIO_ca_CAAMLv6.json
  • OC: https://static.avalanche.report/bulletins/latest/EUREGIO_oc_CAAMLv6.json

The JSON files follow the CAAML v6 standard. Download JSON schema file.
